What is an Upstream?

An upstream represents a backend API service. It defines:

  • Base URL - Where requests are forwarded (e.g., https://api.openai.com)
  • Authentication - API keys, Bearer tokens, or custom headers
  • Connection Settings - Timeouts and connection pooling

Once you create an upstream, you can create routes that target it.

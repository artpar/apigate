System Health

Monitor the health and performance of your APIGate instance.

Health Checks

  • Database - Verifies database connectivity
  • Upstream - Shows configured upstream API status
  • Config - Validates configuration integrity

Status Indicators

  • ● Pass - Component is healthy
  • ● Warn - Component degraded but operational
  • ● Fail - Component is failing

System Metrics

  • Goroutines - Active concurrent processes
  • Memory - Current memory usage
  • Uptime - Time since server started

Troubleshooting

If checks are failing:

  • Check database connection string
  • Verify upstream API is reachable
  • Review server logs for errors

Health API

Programmatic health check endpoints.

Health Endpoint

GET /health

Returns JSON health status. No authentication required.

Response Format

{
  "status": "healthy",
  "checks": [
    {"name": "Database", "status": "pass"},
    {"name": "Upstream", "status": "pass"}
  ]
}

Status Values

  • "healthy" - All checks passing
  • "degraded" - Some checks warning
  • "unhealthy" - Critical checks failing
